# Next.js AI SaaS Boilerplate

Production-ready AI SaaS starter with **Next.js 15**, **Vercel AI SDK 4**, **Supabase**, and **Stripe**.

## Features
- Streaming AI chat (OpenAI, Anthropic, Groq)
- Auth (Supabase + Google OAuth)
- Billing (Stripe subscriptions + usage-based)
- Rate limiting (Upstash Redis)
- Usage dashboard
- Dark mode

## Stack
Next.js 15 · TypeScript · Tailwind CSS · Vercel AI SDK 4 · Supabase · Stripe · Upstash
